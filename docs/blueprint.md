# **App Name**: TuneTwin

## Core Features:

- Song Selection: Allow users to select a song from a dropdown list.
- Similarity Calculation: Initiate the song similarity calculation upon user action.
- Similarity Display: Display the calculated similarity score and related information.

## Style Guidelines:

- Background color: Dark gray (#121212) for a modern, sleek aesthetic.
- Primary color: Vibrant purple (#A06CD5) for the main interactive elements and highlights.
- Accent color: Light purple (#D4A3FF) to add contrast and draw attention to key elements.
- Body and headline font: 'Inter', a sans-serif font, for a clean and modern user experience.
- Use minimalistic icons to represent actions and song characteristics.
- Emphasize clean lines and spacing for a professional presentation.
- Add subtle transition animations when displaying similarity scores.
